# 调试程序的方法

## alert

弹框返回某个值

alert()

## console.log

console.log()

将值返回到控制台，Console 里面



## document.title









```html
<!DOCTYPE html>
<html>
	<head>
		<meta charset="UTF-8">
		<title></title>
		<script type="text/javascript">
			var num01 = 12;
			
			//alert 会把值弹出到页面上，执行顺序是代码的编写顺序
			alert(num01);
			
			
			//console.log() 会把里面的内容输出到控制台
			console.log(num01);
			
			
		</script>
	</head>
	<body>
	</body>
</html>

```
