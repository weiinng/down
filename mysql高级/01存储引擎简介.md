# 存储引擎简介

## innoDB

1.使用命令查看

#看你的mysql现在已提供什么存储引擎：

```mysql
show engines;
```

#查看你的mysql当前默认的存储引擎：

```mysql
show variables like '%storage_engine%'
```

| 对比项   | MySAM                                                  |                            InnoDB                            |
| -------- | ------------------------------------------------------ | :----------------------------------------------------------: |
| 主外键   | 不支持                                                 |                             支持                             |
| 事务     | 不支持                                                 |                             支持                             |
| 行表锁   | 表锁，即时操作一条数据也会锁住整个表，不适合高并发操作 | 行锁，操作时只锁住某一行，不对其他有影响。<br /><font color='red'>适合高并发的操作</font> |
| 缓存     | 只缓存索引，不缓存真实数据                             | 不仅缓存索引，还要缓存真实数据，对内存要求比较高，而且内存大小对性能有决定性的影响。 |
| 表空间   | 小                                                     |                              大                              |
| 关注点   | 性能                                                   |                             事务                             |
| 默认安装 | Y                                                      |                              Y                               |
